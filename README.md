# nikolareljin/tap

This repository is a Homebrew tap for command-line tools published by Nikola Reljin.
It provides prebuilt binaries via Homebrew for fast, reliable installation on macOS
(Apple Silicon and Intel).


## Install

```bash
brew tap nikolareljin/tap
brew install image-view
```

Or in one command:

```bash
brew install nikolareljin/tap/image-view
```

## Available Formulas

### [image-view](https://github.com/nikolareljin/image-view)
CLI image viewer for terminal environments. View images directly in your terminal with automatic scaling and color support.
